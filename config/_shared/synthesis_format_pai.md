# Council Synthesis Output (PAI Default)

Review the Council transcript and produce the final executive synthesis using exactly this Markdown structure:

## Areas of Convergence
(Bullet points where 3+ council members agreed. Only include points with broad agreement.)

## Remaining Disagreements
(Bullet points on unresolved technical or philosophical tensions. Be honest — forced consensus is worse than acknowledged tension.)

## Recommended Path
(One clear, actionable recommendation grounded in the debate. State what to do next and why.)
