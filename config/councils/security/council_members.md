# Security Council Members

Guidance for composing security council experts.

## Perspective Slots (topic-tailored)

| Slot | Purpose | Example Traits |
|------|---------|----------------|
| **Red Teamer** | Adversarial edge cases, active threats | `adversarial, paranoid, edge-case focused` |
| **Pragmatist** | Anti-security-theater, practical engineering | `technical, pragmatic, anti-security-theater` |
| **Risk Officer** | Liability, reputation, user interpretation | `risk management, liability-aware` |
| **Analyst** | Evidence, breach data, industry precedent | `research, analytical, comparative` |

Use `--compose` to auto-generate experts tailored to a specific security scenario.
